# Kinoteca — Catálogo de Cinema Brasileiro

Projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Web, com o objetivo de criar uma plataforma digital voltada exclusivamente para a divulgação, organização e valorização do cinema brasileiro, reunindo filmes, séries, animações, novelas e documentários nacionais em um único ambiente interativo.

Link pro Site: [[https://karineads.github.io/Projeto_de_web_Kinoteca/](https://karineads.github.io/Projeto_de_web_Kinoteca/)]

---

## Objetivo do Projeto

O objetivo principal do projeto **Kinoteca** é aplicar na prática os conhecimentos adquiridos em sala de aula sobre:

- Estruturação de páginas com HTML5  
- Estilização responsiva com CSS3  
- Interatividade com JavaScript  
- Manipulação do DOM  
- Criação de sistema de login  
- Armazenamento de dados no navegador  
- Experiência do usuário (UI/UX)

Além disso, o projeto busca valorizar a cultura audiovisual brasileira por meio da divulgação de produções nacionais.

---

## Tecnologias Utilizadas

- **HTML5** — Estrutura do site  
- **CSS3** — Estilização e layout  
- **JavaScript** — Lógica, interações, login, listas e comentários  
- **LocalStorage** — Armazenamento de usuários, listas e comentários  
- **Git & GitHub** — Controle de versão e repositório  

---

## Funcionalidades do Sistema

- Catálogo dividido por categorias:
  - Filmes
  - Animações
  - Documentários
  - Novelas
  - Séries
- Sistema de **login e cadastro com usuário único e senha**
- Perfil do usuário com:
  - Lista de filmes para assistir
  - Lista de filmes já assistidos
  - Histórico de comentários
- Sistema de **avaliação de 0 a 5 estrelas**
- Comentários por filme
- Exclusão de:
  - Filmes da lista
  - Filmes assistidos
  - Comentários
- Barra de pesquisa
- Pop-up de detalhes do filme com trailer
- Interface responsiva e interativa

---

## Metodologia de Desenvolvimento

O projeto foi desenvolvido de forma incremental, iniciando pela estrutura básica em HTML, seguido da estilização em CSS, e posteriormente da implementação das funcionalidades em JavaScript. Os recursos de login, avaliação, listas e comentários foram implementados utilizando o armazenamento local do navegador (LocalStorage).

Foram aplicados conceitos de:
- Programação orientada a eventos
- Manipulação do DOM
- Validação de dados
- Organização de código
- Boas práticas de interface

## Limitações Atuais

- O sistema não utiliza banco de dados externo, apenas LocalStorage
- O login não possui criptografia de senha
